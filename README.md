Tarowehh adalah aplikasi website kuliner UMKM Keripik Talas berbasis **Vue.js** untuk melihat menu makanan, melakukan pemesanan, dan melihat riwayat transaksi.

## Persiapan & Instalasi
Pastikan sudah menginstall:

- Visual Studio Code
